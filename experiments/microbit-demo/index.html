Info

This app charts accelerometer data from the BBC micro:bit. Note that you must run this app in the Evothings Client app, or build a Cordova app with the plugin com.evothings.ble. A device with support for Bluetooth 4.0 and Android 4.3+ or iOS 7+ is required.

“BBC” and “micro:bit” are trade marks of the BBC.

How to use

Activate the micro:bit and tap the Start button. If everything works the device is discovered and accelerometer data is plotted to the canvas. Shake your micro:bit to see the plot change.

Troubleshooting

On Android, there are sometimes problems with that BLE stops working when switching between apps or doing multiple reloads. In this case restart the Evothings Client app. Additionally, you can use the Evothings Workbench Tools to see any error messages that may help in your investigation.
